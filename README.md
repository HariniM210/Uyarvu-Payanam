# Uyarvu-Payanam 🚀

**Uyarvu Payanam** ("Ascending Journey") is a professional, data-driven career guidance platform designed for students across Tamil Nadu (Class 5 to 12). 

## 🎯 What We Do
We bridge the information gap in career guidance by providing a dynamic ecosystem where students can explore:
- **Milestone Roadmaps**: Guidance for after 5th, 8th, 10th, and 12th standards.
- **College & Course Discovery**: Verified database of Tamil Nadu colleges and institution-specific course offerings.
- **Scholarship Alerts**: Real-time notifications for government and private scholarships.
- **Data-Driven Insights**: TNEA cutoff analysis and career growth trends.

See the full [Project Abstract](./PROJECT_ABSTRACT.md) for more details.

## 🛠️ Tech Stack
- **Frontend**: React.js, Vite, Vanilla CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB Atlas
- **Icons**: Lucide/Feather Icons
